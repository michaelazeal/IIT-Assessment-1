Project Idea

Mobile Application - Horse Quiz

The project I would like to work on is an app for Pony Clubs and Horse Riding Schools which teaches riders the information required to sit their Equestrian Australia Certifications. It would would be a educational app similar to the ‘mimo’ coding quiz app.

The application would have both a educational component with different levels to complete before a quiz was presented to the user. Each stage would follow the protocols set out by Equestrian Australia regarding their minimum level of knowledge required to sit each Certificate.
The application would have two components, the learning portion with levels to complete and the quizzes.

Learning levels
The learning levels would be made up of chapters of educational materials teaching the user about a specific component of horse husbandry, riding or general information.

Quiz
The application would hold thousands of quiz questions and answers that have been grouped into levels for users to complete as they go through the learning content.
The user would be able to join groups, such as be in teams with members from their pony club and be able to see the results of others in their region to foster friendly competition.
